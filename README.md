FaceFun 脸趣 · AI人脸隐私保护工具
 
项目介绍
 
一款面向毕业季、企业团建、聚会合影场景的在线AI人脸批量打码Web工具，依托人脸检测算法自动识别图片内所有人脸，提供马赛克、高斯模糊、表情贴纸三种脱敏方案，全程本地浏览器计算，图片不上传服务器，兼顾处理效率与用户隐私安全。
 
解决日常分享合照泄露他人面部隐私痛点：朋友圈、社群发布集体照片前，批量遮挡人脸，合规安全分享，支持批量图片导入、高清无损导出，适配手机/电脑全端访问。
 
核心功能
 
- ✅ AI智能人脸检测：单张图识别多张人脸，全自动标记无需手动框选
​
- ✅ 三种脱敏模式：马赛克遮挡 / 高斯模糊 / Emoji表情覆盖
​
- ✅ 本地离线运算：图片仅在浏览器处理，不上传云端，隐私零泄露
​
- ✅ 批量快速处理：多图批量导入，一键批量应用脱敏效果
​
- ✅ 高清无损导出：保存原图画质，无压缩失真
​
- ✅ 全平台兼容：Web在线运行，适配 Android / iOS / Windows / macOS，无需下载客户端
 
适用场景
 
1. 毕业合照：毕业季晒学士服合影，批量遮挡同学脸部
​
2. 公司团建：企业活动合照，保护同事面部隐私
​
3. 线下聚会：亲友活动合影，社交平台安心发布
​
4. 活动纪实：线下活动拍摄素材，批量脱敏后对外分发
 
技术亮点
 
1. 前端本地人脸推理，无后端图片存储，从根源保障隐私
​
2. 轻量化算法，移动端低配置设备也能流畅识别
​
3. 轻量化Web应用，开箱即用，无环境依赖，打开网页即可使用
​
4. 实时预览脱敏效果，自由切换遮挡样式，操作极简
 
使用方式
 
1. 打开在线网页，上传单张/多张合照图片
​
2. 工具自动检测图内全部人脸
​
3. 选择马赛克/模糊/Emoji三种打码方案，实时预览效果
​
4. 一键批量导出高清无损图片，本地保存使用
